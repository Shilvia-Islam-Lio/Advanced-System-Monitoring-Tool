**Advanced System Monitoring Tool**
This is a Python-based system monitoring tool that shows real-time CPU, memory, disk, and process information.

## Features

* Real-time CPU monitoring
* Memory and disk usage
* Process tracking
* Live CPU graph
* Alert system
* Logging system
* Pause and resume feature
  
## Technologies Used

* Python
* psutil
* matplotlib
* colorama
* 
## How to Run

1. Install Python
2. Install libraries:
   pip install psutil matplotlib colorama
3. Run:
   python full_monitor.py
   
## Description
This project works like a mini task manager and helps to understand system performance in real time.
